# Programação Básica em R

Este repositório reúne exercícios e exemplos práticos desenvolvidos em linguagem **R**, com foco no aprendizado dos conceitos fundamentais de programação e manipulação de dados.

## Conteúdos abordados:
- Estruturas básicas (variáveis, vetores, listas)
- Condicionais (`if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Funções
- Manipulação de dados
- Leitura e escrita de arquivos (`.csv`, `.xlsx`)
- Introdução a pacotes como `tidyverse`

## Objetivo:
Consolidar os conhecimentos iniciais em programação utilizando R, com aplicações práticas voltadas para análise de dados.

## Tecnologias utilizadas:
- R
- RStudio
